# Multi-Tool AI Chatbot

A full-stack AI chatbot built with **Python**, **LangChain**, and **Ollama (Llama 3.2)** that supports:

- **Real-time and historical weather queries**
- **Movie search and discovery**
- **Bilingual movie title handling**
  - **Chinese titles → Aiyifan**
  - **English titles → GoMovie**

Users can ask questions like:
- “What’s the weather in Toronto today?”
- “What will the weather be in Windsor this weekend?”
- “Show me historical weather in Vancouver from March 1 to March 5.”
- “Find the Chinese movie 流浪地球”
- “Search for Interstellar”

## Features
- Natural language intent classification
- City + date/date-range extraction
- Relative date support (today, tomorrow, next 3 days, this weekend)
- Real-time + historical weather lookup
- Bilingual movie search routing
- Multi-source API orchestration
- Time-Check node for improved temporal accuracy
- Full-stack deployment (GitHub Pages + VPS backend)

## Tech Stack
