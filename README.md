README Intro
AI Weather Chatbot

A full-stack AI weather chatbot built with Python, LangChain, and Ollama (Llama 3.2) that supports real-time and historical weather queries through natural language.

Users can ask questions like:

“What’s the weather in Toronto today?”
“What will the weather be in Windsor this weekend?”
“Show me the historical weather in Vancouver from March 1 to March 5.”

The application uses:

GitHub Pages for the frontend
Python API backend hosted on a VPS
LangChain for structured parsing and workflow orchestration
Ollama (Llama 3.2) for local LLM inference
Weather APIs for real-time and historical data
Key Features
Natural language weather requests
City + date/date-range extraction
Relative date support (today, tomorrow, next 3 days, this weekend)
Historical weather lookup
Real-time weather API integration
Time-aware “Time-Check” node to reduce date hallucinations
Fully deployed full-stack architecture
